# Workspace Reservation System

A Single Page Application (SPA) for managing workspace reservations within a company.

---

## Description

The system allows employees to reserve shared workspaces such as meeting rooms, private offices, coworking spaces, and auditoriums. It includes role-based access control, session persistence, and full CRUD operations for reservations.

---

## Technologies Used
- JavaScript
- JSON Sever
- HTML & CSS
